# Changelog

## [1.1.0] - 2026-07-16

### Added
- **Redis Streams Event Bus**: Decentralized microservice architecture messaging tick and signal streams.
- **Market Data Websocket Service**: Direct pooler streaming Greeks and market depths for 11 brokers, featuring AngelOne adapter.
- **Separate Risk Engine**: Position checks (Kelly and ATR formulas) and kill switch circuit breakers.
- **Validation Lab**: Walk-forward splits and Monte Carlo random sequence analyzers.
- **AI Copilot NLP Engine**: Direct conversational performance audits.
- **Mission Control UI**: Interactive status grids, allocated exposures, and network latencies.

## [1.0.0] - 2026-07-15

### Added
- **API gateway & authentication**: Express server protecting routes via JWT keys.
- **SQL database integration**: prisma schemas mapping strategies, backtests, and trade logs.
- **Backtesting & simulated replay engines**: Candle-by-candle simulation loops.
- **Web Terminal**: Next.js 15 Tailwind charts dashboard.
