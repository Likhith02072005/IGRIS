# GitHub Discussions Setup Guide

To configure GitHub Discussions for IGRIS:

1. Navigate to the repository settings page on GitHub: `https://github.com/Likhith02072005/IGRIS/settings`
2. Under the "Features" section, check the box next to **Discussions**.
3. Create the following standard discussion categories:
   - **Announcements**: Platform upgrades and release notes.
   - **General**: Open discussions.
   - **Q&A**: Configuration help.
   - **Ideas**: Strategy concept pitches.
   - **Show and Tell**: Showcase custom strategy performance.
